3 API Endpoints

POST /api/v1/generate

Генерация изображения

Создает новое изображение на основе текстового описания

Параметры запроса:

{
  "prompt": "beautiful sunset over mountains",
  "width": 1024,
  "height": 1024,
  "steps": 28,
  "cfg_scale": 7.0,
  "model": "runware:101@1"
}

Пример ответа:

{
  "success": true,
  "job_id": "abc123...",
  "status": "processing",
  "cost": {{ cost_per_generation }},
  "balance_remaining": 100.00
}
GET /api/v1/status/{job_id}

Проверка статуса генерации

Получает текущий статус задачи генерации

Пример ответа (в процессе):

{
  "success": true,
  "status": "processing",
  "progress": 45
}

Пример ответа (завершено):

{
  "success": true,
  "status": "completed",
  "image_url": "https://example.com/image.jpg",
  "prompt": "beautiful sunset over mountains"
}
GET /dashboard/api/check-balance/

Проверка баланса

Получает информацию о текущем балансе и статистике использования

Пример ответа:

{
  "balance": 100.00,
  "total_spent": 50.00,
  "total_requests": 125,
  "total_generations": 5
}

4 Примеры кода

Python 

import requests

API_TOKEN = "your_api_token_here"
API_URL = "{{ request.scheme }}://{{ request.get_host }}/api/v1"

headers = {
    "Authorization": f"Bearer {API_TOKEN}",
    "Content-Type": "application/json"
}

# Генерация изображения
response = requests.post(
    f"{API_URL}/generate",
    headers=headers,
    json={
        "prompt": "beautiful sunset over mountains",
        "width": 1024,
        "height": 1024
    }
)

result = response.json()
job_id = result["job_id"]

# Проверка статуса
status_response = requests.get(
    f"{API_URL}/status/{job_id}",
    headers=headers
)

print(status_response.json())

JavaScript / Node.js 

const API_TOKEN = 'your_api_token_here';
const API_URL = '{{ request.scheme }}://{{ request.get_host }}/api/v1';

// Генерация изображения
const response = await fetch(`${API_URL}/generate`, {
    method: 'POST',
    headers: {
        'Authorization': `Bearer ${API_TOKEN}`,
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        prompt: 'beautiful sunset over mountains',
        width: 1024,
        height: 1024
    })
});

const result = await response.json();
const jobId = result.job_id;

// Проверка статуса
const statusResponse = await fetch(
    `${API_URL}/status/${jobId}`,
    {
        headers: {
            'Authorization': `Bearer ${API_TOKEN}`
        }
    }
);

const status = await statusResponse.json();
console.log(status);
